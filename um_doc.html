Informatica Ultra Messaging Documentation Sets


© Copyright Informatica LLC 2004-2019.

The Ultra Messaging doc set has undergone various evolutions over time. Starting with UM version 6.11, we combine the doc for the main messaging products (UMS, UMP, UMQ, DRO) into a single set. Earlier UM versions had separate doc sets for each product.

PRO TIP: when you bring up the main page it has links to individual manuals along the left. Right-click on the desired manual and open in a separate tab.

NOTE: for older UM versions, contact Informatica Support.

All content is copyright Informatica.

To see an expanded "live" version of the latest UM version's doc, go here: https://ultramessaging.github.io/currdoc/

To see an expanded "live" version of the latest "Desktop Services" (UMDS) version's doc, go here: https://ultramessaging.github.io/currumdsdoc/
