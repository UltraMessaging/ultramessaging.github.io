          
	  Ultra Messaging(R) Documentation
     --------------------------------------------------------------

     --------------------------------------------------------------

	Beginning with Ultra Messaging Core (UMS, UMP, UMQ) 
	Version 5.2, the Ultra Messaging documentation does not 
	accompany the product package. 
	
	You must download the UM documentation separately. 
	
	Please return to the Informatica Ultra Messaging Download page
	(http://www.29west.com/cgi-bin/download). 
	
	The documentation package can be found in the same 
	directory as the product binary package.
