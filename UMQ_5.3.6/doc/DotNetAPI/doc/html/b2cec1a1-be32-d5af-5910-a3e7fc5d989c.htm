scheduleTimer Method (milliseconds, cb, cbArg)
Namespaces ► com.latencybusters.lbm ► LBMContext ► scheduleTimer(Int64, LBMTimerCallback, Object)
Creates a timer associated with this context that expires with specified delay, and retains an internal reference to it. This prevents the timer from being reclaimed by the garbage collector before it expires.

Parameters
milliseconds (Int64)
Number of milliseconds until the timer expires
cb (LBMTimerCallback)
Callback implementing the LBMTimerCallback delegate interface
cbArg (Object)
Callback object passed as token to callback interface
Return Value
LBMTimer object
Remarks
See createTimer(Int64) to create a timer that will be reclaimed by the garbage collector in the normal manner once it goes out of scope.
