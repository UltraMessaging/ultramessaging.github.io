scheduleTimer Method (milliseconds)
Namespaces ► com.latencybusters.lbm ► LBMContext ► scheduleTimer(Int64)
Create an LBM Timer object and retains an internal reference to it. This prevents the timer from being reclaimed by the garbage collector before it expires.
Declaration Syntax
C#
public LBMTimer scheduleTimer(
	long milliseconds
)
Parameters
milliseconds (Int64)
Number of milliseconds until the timer expires
Return Value
LBMTimer object
Remarks
See createTimer(Int64) to create a timer that will be reclaimed by the garbage collector in the normal manner once it goes out of scope.
