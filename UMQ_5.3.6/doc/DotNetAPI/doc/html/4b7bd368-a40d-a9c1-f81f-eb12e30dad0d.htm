cbInfo (UMERecoverySequenceNumberCallbackInfo)
Recovery sequence number info. See UMERecoverySequenceNumberCallbackInfo
Return Value
This callback should always return 0. The lowest requested sequence number should be set, if desired, by setLowSequenceNumber(UInt32) before returning.
Remarks
Sets the lowest sequence number to be requested during recovery.
